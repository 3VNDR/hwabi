# hwabi
An attempt to familiarize myself with Zig by building a v95 Maplestory server! Very early on and a total work in progress

## Requirements

Before building and running Hwabi, make sure you have the following installed:

- [Zig](https://ziglang.org/) 0.16
- [PostgreSQL](https://www.postgresql.org/) 14 or newer
